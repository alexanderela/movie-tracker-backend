To get user data for a specific station, use the URL:

https://movie-tracker-backend-ae.herokuapp.com/api/v1/stations/:station_id

in your fetch request, which returns a response body containing an array which contains the specified station object.

Example URL:


            https://movie-tracker-backend-ae.herokuapp.com/api/v1/stations/1

POST Endpoints

To add a user to the database, use the URL:

https://movie-tracker-backend-ae.herokuapp.com/api/v1/stations/

in your fetch request, which returns a response body with an id and a message.

Example Request:


            {
              "station_name": "Chris' Awesome Station",
              "station_phone": "555-555-5555",
              "street_address": "123 17th",
              "city": "Denver",
              "state": "CO",
              "zip_code": 80220,
              "latitude": 39.75,
              "longitude": -105,
              "intersection_directions": "unknown",
              "access_days_time": "24 hours daily"
            }

Example Response:


            {
              "id": 35,
              "message": "Station \"Chris' awesome station\" successfully created!"
            }

POST Endpoints

To add a favorite to the database, use the URL:

https://movie-tracker-backend-ae.herokuapp.com/api/v1/users/favorites/new

in your fetch request, which returns a response body with an id and a message. 


            https://movie-tracker-backend-ae.herokuapp.com/api/v1/users/favorites/new

Example Request:


            {
				      "movie_id": 3,
				      "user_id": 1,
				      "title": "The Mule",
				      "poster_path": "https://image.tmdb.org/t/p/w500//bkc4AY6FTa3yNqrshE9b1elDzPt.jpg",
				      "release_date": "12/14/2018",
				      "vote_average": 6.3,
				      "overview": "A 90 year old horticulturalist and Korean War veteran is caught transporting $3 million worth of cocaine through Illinois for a Mexican drug cartel."
            }

Example Response:


            {
              "id": 103,
              "message": "Cafe "Chris' awesome cafe" successfully created!"
            }

DELETE Endpoints

To delete a specific favorite, use the URL:

https://movie-tracker-backend-ae.herokuapp.com/api/v1/users/:id/favorites/:movie_id

in your fetch request, which returns a response body with an id and a message.

Example URL:


            https://movie-tracker-backend-ae.herokuapp.com/api/v1/users/1/favorites/3

Example Response:


              {
                "id": 3,
                "message": "Favorite movie 3 has been deleted."
              }
